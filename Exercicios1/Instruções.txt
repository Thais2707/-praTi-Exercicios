Inicialemnte é necessário conferir se o node esta corretamente instalado, para isso, abra o terminal e digite "node -v", caso esteja instalado, aparecerá a versão utilizada, caso não apareça, baixe e instale o Node de um site oficial. Em seguida,  será necessária a instalação do prompt-sync no Visual Studio Code, no terminal do proprio programa digite "npm install prompt-sync", e após feita a instalação, será possivel executar os programas. Selecione todos os arquivos .js enviados, clique com o botão direito e selecione "abrir com o code".
